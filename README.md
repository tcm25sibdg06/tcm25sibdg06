# tcm25sibdg06
O objetivo do nosso trabalho é desenvolver um sistema de gestão de base de dados que permita registrar e organizar a informação de forma eficiente, melhorando a comunicação com os clientes e a gestão interna da clinica, ou seja, automatizar e organizar o funcionamento da clínica veterinária, melhorandoa gestão das consultas,o controlo dos animais e donos, o trabalho dos profissionais, a qualidade do atendimento a eficiência geral da clínica organizando de forma eficiente as marcações realizadas pelos donos.
